# Frontend Design Structure

## Overall Structure

    Frontend Technology
      React
      Tailwind / CSS (style)
      Typescript
      Apollo-client (Connection)

    Work Load
      Plan (10%)
      Design (55%)
          Input/Output
          Algorithm Steps
          Education Section
          Uniqueness (Animation/Coloring/Etc)
      Algorithms (30%)
          Code
          Test (jest)
      Backend Connections (5%)

## Page Structure

    Landing
    Tools
        Design page
        code algorithm(s)
        code frontend

## HOW TO RUN

    Frontend:
        run 'yarn' to get dependencies
        run 'yarn start' to start react app

## Performing Individual Test

    yarn test
    or
    tsc filename.tsx
    node filename.js
